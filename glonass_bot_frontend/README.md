# Admin Panel - Бот рассылок

Административная панель для управления ботом рассылок с поддержкой пользователей и постов.

## Требования

- Node.js 18+ (для локальной разработки)
- Docker и Docker Compose (для контейнеризации)
- npm или yarn

## Быстрый старт

### 1. Клонирование репозитория

```bash
git clone <repository-url>
cd admin-panel
